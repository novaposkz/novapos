Условные обозначения

Описание пиктограмм и условных обозначений в кассовой программе GBS.Market

Главная > База знаний > Условные обозначения

В кассовой программе GBS.Market на кнопках используются пиктограммы для обозначения действий. Ниже перечислены основные используемые обозначения.

Общие

Добавить

Добавление новой записи. Например, добавление товара в каталоге товаров.

Редактировать

Редактирование выбранной записи. Например, редактирование товара в каталоге

Изменить количество

Изменение количества, например, для товара в процессе продажи

Комментарий

Например, комментарий к товару или продаже

Печатать

Печать документа для выбранной записи

Создать копию

Например, в каталоге товаров создание копии выбранных товаров

Объединить

Объединение записей, например, объединение товаров в каталоге

Удалить

Удаление позиции из списка. Например, удаление товара

Информация

Информация о выбранной записи. Например, в журнале продаж открывает карточку продажи

Возврат

Возврат продажи в журнале продаж

Внести платеж

Внесение платежа. Например, платеж по задолженности в списке должников

Перенести позицию

Перенос выбранных позиций в другой список

Перенести все позиции

Перенос всех позиций в другой список

Параметры

Открывает дополнительные параметры фильтрации/поиска

Еще

Открывает меню с дополнительными действиями

Корзина/чек

Скидка на позицию

Установить скидку на один или несколько выбранных товаров

Скидка на весь чек

Установить скидку на все позиции в чеке

Взвесить

Открыть окно для взвешивания товара

Режим "Магазин"

Поиск товаров

На главной форме - открывает окно поиска со всеми товарами

Избранные товары

Список избранных товаров на главной форме

Режим "Кафе"

Домой

В режиме "Кафе" возвращает в корень меню категорий

Контакт

Выбор контакта (покупателя)

Активные заказы

В режиме "Кафе" открывает список активных заказов

Выход

Выход из режима "Кафе" в режим "Магазин"

Открыть "Кафе"

Переход из режима "Магазин" в режим "Кафе"

Завершение работы

Закрыть программу

Окно входа

Вход

Вход сотрудника в программу

Выход

Выход сотрудника из программы

Информационные сообщения

Информация

Информационное сообщение

Вопрос

Сообщение с вопросом

Успех

Сообщение об успешном выполнении действия

Предупреждение

Предупреждение при выполнении действия

Ошибка

Сообщение об ошибке
