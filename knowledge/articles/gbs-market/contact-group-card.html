Карточка группы контактов

Описание внешнего вида карточки группы контактов

Главная > База знаний > Карточка группы контактов

Категория (группа) контактов предназначена для объединения контактов, имеющих схожие характеристики.

В меню главной формы нажмите Контакты – Группы контактов, а затем создайте новую или выберите для редактирования, чтобы открыть карточку группы.

Свойства группы контактов

Название группы

В качестве названия группы можно ввести короткое описание категорий контактов, например:

  • Постоянные клиенты
  • Неплательщики
  • Поставщики
  • Скидка 5%

Колонка цен

Колонка цен – это дополнительная цена, по которой будет продаваться товар при выборе покупателя из данной группы.

Например, оптовым покупателям можно отпускать товары по отдельной колонке цен.

Максимальная сумма долга

Максимальная сумма долга позволяет ограничить продажи в долг для контактов из такой группы.

Например, если для группы указано ограничение в 1000 рублей, то покупатель из такой группы не сможет приобрести товаров в долг больше, чем на 1000 рублей.

Ограничение происходит по общей сумме задолженности покупателя.

Для того чтобы отключить ограничение, оставьте поле пустым.

Полезные материалы

Личная скидка

Размер личной скидки для покупателей этой группы. Скидка будет предоставлена при выборе покупателя из группы в момент продажи.

Полезные материалы

  • Видео-урок "все о скидках"

Поставщик

Включите опцию, чтобы контакты из этой группы отображались в списке поставщиков.

Запретить использовать бонусы

При включении данной опции использование баллов контактами из группы будет запрещено.
