Описание раздела настроек "Оборудование – Доп. принтеры". Открыть настройки можно из главного окна программы Файл – Настройки.

Назначение дополнительных принтеров

Дополнительные принтеры могут использоваться для печати чеков заказов в режиме "Кафе". Например, можно настроить несколько дополнительных принтеров – для кухни, холодного цеха и мангальщика.

При сохранении заказа в "Кафе" будут напечатаны чеки, согласно настройками принтера.

Отправлять данные на дополнительные принтеры

Если опция включена, то программа будет отправлять на печать чеки, согласно правилам каждого принтера.

Управление списком принтеров

Добавить

Кнопка "Добавить" Нажмите кнопку, чтобы добавить принтер в список

Добавление доп. принтера для печати чеков

Карточка доп. принтера и его настройки

В карточке доп. принтера необходимо:

  • выбрать доступных для печати принтер чеков
  • указать категории, товары из которых будут отправляться на этот принтер
  • выбрать шаблон, по которому будут формироваться чеки для указанного принтера
  • активировать принтер

Редактировать

Кнопка "Редактировать" Нажмите кнопку, чтобы изменить выбранный принтер

Выберите в списке принтер и нажмите кнопку "редактировать", чтобы внести изменения в карточке доп. принтера и его настроек.
