Описание раздела

Описание раздела настроек "Оборудование – Весы". Открыть настройки можно из меню главного окна: Файл – Настройки – Оборудование – Весы.

В данном разделе настроек можно выполнить конфигурацию подключения к торговым весам и к весам с печатью этикеток, которые поддерживает GBS.Market.

Полезные материалы

  • Поддерживаемые торговые весы
  • Поддерживаемые весы с печатью этикеток

Внешний вид окна настроек Оборудование - Весы

Весы

Опции для настройки торговых весов

Тип весов

Тип весов, подключаемых к программе.

Доступны варианты:

  • CAS
  • WIntec
  • Massa-K
  • Атол/ШТРИХ-М
  • и другие

Полезные материалы

  • Полный список поддерживаемых торговых весы

Настройки подключения

Настроить

При нажатии на кнопку "Настроить" будет отображено окно для конфигурации подключения весов к программе. Вид настроек может зависеть от типа выбранных весов.

Например, может быть открыто окно настройки подключения через COM-порт, если весы работают через указанный интерфейс.

Настройки COM-порта для подключения весов

Проверка связи

При нажатии на кнопку "проверка" связи будет отображено окно, показывающее вес, находящийся на весах.

Проверка веса

Показывать кнопки управления весами: Tara, Zero в форме ввода кол-ва

Важно

Данные кнопки могут не работать на некоторых моделях весов

Если опция включена, то в окне ввода кол-ва товара будут отображаться дополнительные кнопки:

  • Tara (тара) – установка значения тары
  • Zero (ноль) – установка значения "ноль"

Отображение дополнительных кнопок в форме ввода кол-ва

Показывать кнопку проверки веса в корзине

Если опция включена, то в корзине будет отображена дополнительная кнопка для проверки веса. Нажав данную кнопку можно быстро взвесить любой товар, без добавления его в чек.

Весы с печатью этикеток

Набор настроек для конфигурации подключения к весам с печатью этикеток

Тип весов

Тип весов, подключенных к программе.

Доступны варианты совместимых весов:

  • Штрих-М
  • CAS
  • Rongta

Полезные материалы

  • Перечень поддерживаемых весов с печатью этикеток
  • Загрузка PLU-кодов и данных о весовых товарах в весы с печатью этикеток
  • Инструкция по настройке подключения к весам с печатью этикеток Rongta

Настройки подключения

Настроить

При нажатии на кнопку "Настроить" будет отображено окно для конфигурации подключения весов к программе. Вид настроек может зависеть от типа выбранных весов.

Например, может быть открыто окно настройки подключения через COM-порт, если весы работают через указанный интерфейс.

Настройки COM-порта для подключения весов

Поле PLU

В списке можно выбрать одно из доп. полей с типом "целое число", которое используется для заполнения PLU (price-look up) кода товаров.
