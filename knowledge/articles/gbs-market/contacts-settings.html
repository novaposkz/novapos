Настройки программы, связанные с контактами (клиентами, покупателями)

Описание раздела настроек "Контакты". Открыть настройки можно из главного окна программы Файл – Настройки.

Настройка доп. полей

Раздел настроек, в котором можно создавать и редактировать дополнительные поля (свойства) для контактов.

Полезные материалы

  • Видео-урок по настройке и использованию дополнительных полей

Синхронизация контактов

Опция отвечающая за синхронизацию контактов между торговыми точками.

Доступны варианты:

  • Нет – синхронизация отключена
  • Файловый обмен – синхронизация включена

При включенном файловом обмене выгружается следующая информация:

Синхронизируемые данные:

  • Данные контакта (ФИО, телефон, email, доп. поля и т.п.)
  • Сумма покупок
  • Сумма баллов
  • Сумма задолженности

Полезные материалы

  • Описание работы файловой синхронизации контактов

Подтверждение действий

Важно

Блок настроек доступен только в случае, когда включена интеграция с сервисами СМС-рассылок. Обратите внимание, что доступность и работоспособность выбранных вариантов доставки кодов зависит от поставщика услуг.

Полезные материалы

  • Подробнее о настройке отправки кодов подтверждения можно узнать в статье

При списании бонусных баллов

Опция определяет, каким образом будет происходить подтверждение списания бонусов при их использовании в момент оплаты.

Доступны варианты:

  • не запрашивать подтверждение - при списании бонусов не будет происходить запрос на подтверждение
  • отправлять код в СМС - код подтверждения будет отправлен через СМС
  • отправлять код в Viber - код подтверждения будет отправлен в Viber
  • отправлять код в WhatsApp - код подтверждения будет отправлен в WhatsApp
  • осуществлять звонок - будет осуществлен звонок покупателю с проверочного номера. Кодом проверки обычно служат последние 4 цифры номера

При покупке в долг

Опция определяет, каким образом будет происходить подтверждения номера телефона покупателя при оформлении продажи в долг.

Доступны варианты:

  • не запрашивать подтверждение - при оформлении продажи в долг не будет происходить запрос на подтверждение
  • отправлять код в СМС - код подтверждения будет отправлен через СМС
  • отправлять код в Viber - код подтверждения будет отправлен в Viber
  • отправлять код в WhatsApp - код подтверждения будет отправлен в WhatsApp
  • осуществлять звонок - будет осуществлен звонок покупателю с проверочного номера. Кодом проверки обычно служат последние 4 цифры номера
