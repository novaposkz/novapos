Установка шаблонов документов

Процесс установки шаблона документов

Главная > База знаний > Установка шаблонов

GBS.Market позволяет устанавливать шаблоны (печатные формы) для документов.

Формат шаблонов

Шаблоны документов – это файлы отчетов FastReport.

Важно!

Попытка открыть файл шаблона двойным кликом, скорее всего, не приведет к какому-либо результату.

Для установки шаблона следуйте инструкции.

Процесс установки шаблона

Для установки шаблонов документов выполните следующие действия:

Откройте настройки программы

Откройте настройки GBS.Market: Файл-Настройки

Откройте папку с шаблонами

Перейдите на вкладку "Внешний вид" и нажмите "Папка с шаблонами"

Скопируйте файл шаблона

Откроется окно проводника с папкой, в которой находятся шаблоны.

Ознакомьтесь с перечнем папок ниже и откройте ту папку, которая соответствует устанавливаемому шаблону, и скопируйте файл шаблона в открытую папку.

Пример

Например, для установки шаблонов ценника, необходимо скопировать файл в папку PriceTags

Проверьте результат

Попробуйте напечатать документ, для которого вы установили шаблон. Если все сделано корректно, то установленный шаблон появится в списке для выбора.

На скриншоте ниже видно, что был установлен 1 шаблон для печати ценников.

Примечания

Совместимость шаблонов

  • Шаблоны могут быть несовместимы между собой.
    Например, шаблон входящей накладной не будет совместим с документами, печатаемые при продаже
  • Некоторые шаблоны могут быть совместимы.
    Например, шаблоны ценников и этикеток совместимы и взаимозаменяемы

Количество шаблонов

  • Может быть установлено неограниченное количество шаблонов
  • Для любого типа документов количество шаблонов не ограничено

Краткая инструкция

  1. Откройте Файл → Настройки
  2. Перейдите на вкладку "Внешний вид"
  3. Нажмите "Папка с шаблонами"
  4. Откройте нужную папку для типа документа
  5. Скопируйте файл шаблона в папку
  6. Проверьте результат при печати документа

Полезные советы

  • Убедитесь, что файл шаблона имеет правильный формат FastReport
  • Проверьте совместимость шаблона с типом документа
  • Создайте резервную копию существующих шаблонов перед установкой новых
  • Тестируйте новые шаблоны на пробных документах
