Контакты - Группы контактов

Описание раздела Контакты - группы контактов, в котором происходит управление списком групп покупателей и поставщиков

Главная > База знаний > Контакты - Группы контактов

Группы контактов – это справочник категорий (групп) контактов.

Разделение контактов на группы позволяет определить некоторые свойства для входящих в эти группы контактов:

  • Разделение на покупателей и поставщиков
  • Установка максимально допустимой суммы задолженности
  • Выделение группы оптовых покупателей с определенной колонкой цен
  • Присвоение скидки для группы постоянных покупателей

Связанные материалы

  • Инструкция по добавлению поставщиков

Полезное видео

  • Видео-урок о настройке дисконтной системы скидок

Открыть форму поиска можно, например, с главной формы, нажав в меню Контакты – Группы контактов.

Информация

Доступ к списку групп контактов, а так же отдельные действия с записями могут быть ограничены правами доступа.

Полезные материалы

  • Видео-урок по настройке прав доступа

Блок "Поиск"

Блок поиска в этом разделе состоит только из поля поиска, предназначенного для ввода поискового запроса, по которому будут фильтроваться группы контактов в списке. Фильтрация происходит только по полю "Название".

Список групп контактов

В списке групп контактов отображаются записи, соответствующие критериям поиска. Если поисковый запрос не указан, то будут отображены все записи, имеющиеся в базе данных.

Добавить

Нажмите на кнопку "добавить", чтобы создать новую группу контактов. Будет открыта карточка группы контактов с возможностью указать информацию о новой группе.

Редактировать

Чтобы изменить группу контактов, выберите необходимый элемент в списке и нажмите кнопку "редактировать"

Откроется карточка группы контактов для редактирования.

Удалить

Выберите одну или несколько групп из списка, чтобы удалить их из справочника. После удаления к этим группам невозможно будет присвоить контакты.

Важно

Удаление групп возможно только в том случае, если в группе нет контактов.

Блок итогов

В блоке итогов отображается суммарная информация по количеству отфильтрованных записей
