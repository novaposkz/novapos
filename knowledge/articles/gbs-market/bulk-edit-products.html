Групповое редактирование товаров

Описание процесса массового (группового) редактирования товаров

Главная > База знаний > Групповое редактирование товаров

Групповое редактирование товаров позволяет выполнять массовые действия над списком товаров.

Для выполнения группового редактирования в меню на главной форме выберите Товары – Групповое редактирование товаров.

Также групповое редактирование доступно из журнала поступлений.

Важно

Групповое редактирование - необратимый процесс. Перед выполнением массовых изменений рекомендуем создавать резервную копию.

Информация

Доступ к групповому редактированию может быть ограничен правами доступа.

Видео-урок

  • Видео-урок по настройке прав доступа

Выбор товаров

После перехода в Товары – Групповое редактирование товаров программа отобразит пустой список.

Добавить

Кнопка "Добавить" Нажмите на кнопку, чтобы добавить товар из каталога

Нажмите на кнопку "добавить", чтобы добавить товар из каталога. При нажатии на кнопку откроется окно поиска товаров.

Удалить

Кнопка "Удалить" Нажмите на кнопку, чтобы удалить выбранные товары

Выберите один или несколько товаров, чтобы удалить их из списка для редактирования.

Далее

Чтобы перейти к выбору действий, нажмите кнопку "далее"

Отмена

Чтобы отменить групповое редактирование, нажмите "Отмена". Программа запросит подтверждение для отмены.

Выбор действий

В форме выбора действий отметьте те, которые вы хотите выполнить с добавленными в список товарами.

Переместить в категорию ...

Включите опцию, если необходимо переместить товары в другую категорию.

Выберите категорию, в которую необходимо переместить товары из списка.

Важно!

Необходимо учесть, что не для всех товаров может быть доступна смена категории. Например, для товаров, у которых тип указан как "подарочный сертификат" нельзя изменить на категорию с типом "услуга" и т.п.

Присвоить случайный штрихкод

Включите опцию, если необходимо присвоить случайный штрихкод для товаров из списка.

В качестве префикса для генерируемых штрихкодов будет выставлен префикс из Файл – Настройки – Оборудование – Сканер штрихкода.

Штрихкод будет сгенерирован в формате EAN-13.

Удалить все нулевые остатки

Информация

Опция доступна начиная с версии 6.3.0.1555

Включите опцию, если необходимо удалить нулевые остатки для товаров. Сами товары при этом НЕ будут удалены.

Важно!

Это действие удалит из списка остатков, отображаемых в карточке товара, те, значение "количество" у которых равно нулю.

Если у товара нет других остатков, это может привести к невозможности продать товар в минус, если на момент группового редактирования его остаток был равен нулю.

Используйте опцию с осторожностью.

Изменить розничную цену

Информация

Опция изменения розничной цены доступна начиная с версии 6.3.0.1555

Выберите эту опцию, чтобы изменить текущую розничную цену для товаров из списка.

Полезные материалы

  • Инструкция по настройке автоматического ценообразования

Установить цену равной...

Выберите эту опцию, если на все товары из списка необходимо установить одинаковую цену.

Посчитать из...

Выбор "старой" цены

Выберите, на основании какой цены происходит расчет новой розничной цены.

Розничная цена может быть посчитана на основании:

Закупочная цена

В таком случае для каждого остатка товара будет установлена розничная цена в зависимости от закупочной цены этого остатка.

Важно

В этом случае для одного товара могут образоваться разные розничные цены. Если вы работаете по схеме "один товар - одна цена", используйте пункт "Последняя закупочная цена"

Последняя закупочная цена

При выборе этого варианта все остатки товара будут иметь одинаковую розничную цену, которая будет посчитана из закупочной цены последнего поступления товара.

Розничная цена

В данном случае расчет розничной цены будет происходить на основании текущей розничной цены.

Например, этот вариант подойдет для случаев, когда нужно увеличить цену для всех товаров на 10%.

Выбор оператора изменения

Выберите, каким способом будет изменяться цена

Прибавить сумму...

Выберите этот вариант, если для изменения цены необходимо прибавить к "старой" цене определенную сумму.

Например, можно увеличить цену определенных товаров на 100 рублей.

Умножить на...

Выберите этот вариант, если для изменения цены необходимо изменить цену на определенный коэффициент.

Например, для увеличения цены на 10% процентов необходимо указать значение коэффициента 1,1. Для уменьшения цены на 10% – 0,9.

Округлять до...

Включите опцию, чтобы округлять полученные при переоценке цены. Цены будут округлены кратно указанному коэффициенту.

Например, чтобы округлить цены до рубля, укажите 1.

Изменять только положительные остатки

Включите эту опцию, чтобы изменения цены применились только для тех остатков, значение "количество" у которых больше нуля.

Важно

Если вы практикуете продажу товаров в минус и некоторые товары имеют отрицательные остатки - опцию необходимо оставить выключенной, чтобы переоценка произошла для всех остатков.

После редактирования распечатать...

Включите опцию, если необходимо распечатать ценники или этикетки после изменения розничных цен.

Важно

Не забывайте включать опцию, если необходимо обновить ценники, т.к. сейчас нет истории группового изменения и в последствии придется вручную составлять список товаров для печати ценников, если это не сделано сразу.

Выполнить

Для подтверждения выбранных действий нажмите "Выполнить".

Важно

Групповое редактирование - необратимый процесс. Перед выполнением массовых изменений рекомендуем создавать резервную копию.

Отмена

Чтобы отменить групповое редактирование, нажмите "Отмена". Указанные действий не будут выполнены, произойдет возврат в список товаров.
