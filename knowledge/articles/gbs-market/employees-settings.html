Описание раздела настроек "Сотрудники". Открыть настройки можно из главного окна программы Файл – Настройки.

Сотрудники

Справочник сотрудников, в котором производится добавление и редактирование учетных записей сотрудников организации.

Полезные материалы

  • Видео-урок: настройка сотрудников и установка прав доступа

Группы сотрудников

Раздел для настройки групп сотрудников и прав доступа для них. Нажмите на кнопку, чтобы открыть справочник "Группы сотрудников"

Авторизация

Способ авторизации по умолчанию

Предпочитаемый способ авторизации в программе – пароль или штрихкод.

Не запрашивать авторизацию, если активен только один сотрудник

Если опция включена и в программу выполнил вход только один сотрудник, то при выполнении действий авторизация не будет запрашиваться. Все действия будут выполняться от лица сотрудника, выполнившего вход.

Если у сотрудника недостаточно прав на выполнение действия, то авторизация будет запрошена.

Дополнительно

Закреплять продажу за сотрудником

Если опция включена, то продажа будет закреплена за сотрудником.

Если вход в программу выполнили несколько сотрудников, то при сохранении продажи будет запрошена авторизация.

Сокращать ФИО сотрудника при печати

Если опция включена, то на печатаемых документах вместо "Иванов Петр Сергеевич" будет сокращенный вариант "Иванов П. С."

Отчет продавца

Включите опцию "Отчет продавца", чтобы активировать доступ для сотрудников. В этом отчете продавец сможет отслеживать сумму сделанных продаж, оплату, возвраты. Отчет доступ из меню главной формы: Отчеты - Отчет продавца

Если необходимо настроить, какую информацию отображать, нажмите кнопку "Настройка отчета"

Вы сможете включить или отключить отображение:

  • возвраты - количество возвратов
  • снятие наличных - сумма снятий
  • внесение наличных - сумма внесений
  • сумма в кассе - сумма наличных в кассе для текущей секции
  • сумма в ККМ - сумма наличных, полученная из кассы (ККМ)
  • сумма платежей - оплаченная сумма
  • в т.ч. оплаты по способам - кнопка "Оплачено" становится активной, при нажатии на которую будет отображаться информация о суммах по способам оплаты
  • разрешить выбор даты - включает отображение календаря, где сотрудник может выбрать дату, за которую формируется отчет

История действий

Хранить историю действий в течение...

Количество дней, в течение которых в базе данных будет храниться история действия сотрудников.

История действий

Раздел для просмотра истории действий пользователей. В истории отображаются почти все действия, которые выполняются сотрудниками – просмотр, изменение, удаление данных.
