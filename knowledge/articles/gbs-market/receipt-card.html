Карточка накладной

В кассовой программе GBS.Market доступно управление входящими накладными, через которые происходит оформление поступления товаров от поставщика

Главная > База знаний > Карточка накладной

В кассовой программе GBS.Market доступно управление входящими накладными, через которые происходит оформление поступления товаров от поставщика. Создание и редактирования такого документа выполняется в карточке накладной.

В карточке накладной можно:

  • оформить новое поступление
  • установить количество и закупочные цены для поступивших товаров
  • принять товары из другой торговой точки
  • принять накладную из ЕГАИС
  • создать поступление из Excel
  • создать поступление из УДП (ЭДО)
  • вести учет платежей по накладной в пользу поставщика

Выглядит карточка накладной так, как показано на скриншоте ниже.

Карточка накладной открывается при создании новой накладной из меню главной формы Документы – Новое поступление и при создании или редактировании накладных из журнала поступлений.

Блок "Параметры накладной"

В этом блоке указывается основная информация о входящей накладной, такая как: дата, номер, поставщик, склад и т.д.

Дата

В поле дата указывается фактическая дата поступления товаров по накладной. По умолчанию заполнено текущей датой.

Введите дату вручную или нажмите на иконку календаря, чтобы выбрать значение даты в календаре.

Номер

Поле "номер" содержит номер накладной – это может быть внутренний номер накладной торговой точки или номер, указанный на документе накладной от поставщика.

Поставщик

Нажмите на кнопку, чтобы выбрать поставщика в окне поиска контактов, от которого поступили товары в рамках данной накладной. Значение "поставщик" может быть пустым, если нет необходимости в его заполнении.

Склад

В поле склад необходимо выбрать склад, на который поступают товары при приеме накладной.

Если в настройках (Настройки – Основные) указан только один склад, то значение поля будет заполнено автоматически. Если складов несколько – необходимо будет указать склад.

Важно

Указать склад можно только в момент создания накладной. Изменение склада в процессе редактирования накладной нельзя.

Еще в пути

Опция для установки статуса накладной. Если опция в момент сохранения не включена, то подразумевается, что накладная сохраняется в статусе "принята". Это значит, что товары, поступившие в рамках накладной, будут зачислены на остаток (их количество увеличится согласно накладной).

Если опция включена, то товары на остаток не будут зачислены – их остатки не изменятся.

Важно

Если накладная была сохранена в статусе "Принята", а в процессе редактирования для нее будет изменен статус на "Еще в пути", то остатки товаров будут уменьшены.

Комментарий

В поле "комментарий" может быть указана дополнительная информация по накладной.

Список товаров

На вкладке "Список товаров" отображаются товары, поступившие в рамках входящей накладной.

В списке отображается информация:

  • Штрихкод
  • Категория товара
  • Наименование товара
  • Закупочная цена
  • Кол-во
  • Розничная цена
  • Дополнительные поля товаров, например, артикул, производитель и т.п.

В таблице порядок и перечень столбцов может быть изменен.

Добавить

Кнопка "Добавить" Нажмите кнопку, чтобы добавить в накладную товар.

При нажатии откроется меню для выбора способа добавления товара в накладную.

  • Новый товар – откроется карточка товара для создания нового товара, который будет добавлен в накладную
  • Из каталога – откроется окно поиска товаров, которые есть в базе данных программы
  • Из Excel – откроется окно импорта накладной из Excel-документа
  • Из другой точки - откроется окно выбора накладной для приема товара из другой торговой точки
  • Из УПД (ЭДО) - откроется окно для загрузки накладной из файла УПД (универсального передаточного документа) формата xml

Важно

Пункт меню "Из другой точки" отображается только в том случае, если есть перемещения, отправленные из других торговых точек.

Редактировать

Кнопка "Редактировать" Нажмите кнопку, чтобы изменить цену и кол-во товара

Выберите один или несколько товаров в документе и нажмите кнопку "редактировать", чтобы изменить:

  • закупочную цену
  • закупочную сумму
  • процент наценки
  • кол-во
  • розничную цену

При нажатии кнопки откроется окно:

Кнопка "Перечитать цену" в окне редактирования предназначена для автоматического расчета розничной цены исходя из правил автоматического ценообразования.

Поле ввода процента наценки можно включить в настройках программы, Внешний вид – Показывать процент наценки в карточке накладной

Кнопка для установки предыдущей цены отображается в случае, если в настройках Действия с товарами включена опция "Предлагать предыдущую розничную цену"

Создать копию товара

Кнопка "Создать копию" Нажмите кнопку, чтобы создать копию товара

Выберите товар в списке и нажмите кнопку "создать копию". Программа создаст копию выбранного товара в каталоге, после чего будет открыта карточка товара для редактирования созданной копии. Далее этот товар будет добавлен в накладную.

Удалить

Кнопка "Удалить" Нажмите кнопку, чтобы удалить запись из документа

Выберите один или несколько товаров в накладной, чтобы удалить их из документа. Товары будут удалены только из документа, но останутся в каталоге товаров.

Важно

Если накладная уже была сохранена и находится в статусе "Принята", то удаление товаров из накладной приведет к уменьшению остатков удаленного товара.

Еще

Кнопка "Еще" Нажмите кнопку, чтобы открыть меню с дополнительными действиями

Выберите товар и нажмите кнопку "еще", чтобы показать меню с дополнительными действиями.

Рассчитать розничную цену

При нажатии на пункт меню произойдет пересчет розничных цен для выбранных товаров согласно правилам автоматического ценообразования.

Полезно

Можно выбрать несколько записей в списке для массового изменения розничной цены

Полезное видео

  • Настройка правил автоматического ценообразования

Редактировать товар

Выберите товар для редактирования. Будет открыта карточка товара для изменения его свойств.

Заменить товар

Данный пункт предназначен для замены товара в случае добавления накладной из документа Excel. При нажатии на данный пункт меню будет открыто окно поиска товаров для замены.

Например, при добавлении накладной был создан новый товар, вместо сопоставления с уже существующим. Для того чтобы в каталоге не создавался дубликат – используйте данный пункт меню.

Блок итогов списка товаров

В блоке итогов списка товаров отображается сводная информация по всем позициям в документе.

  • Наименований – кол-во записей в документе
  • Товаров – сумма кол-ва всех товаров в списке
  • На сумму – общая сумма товаров по закупочной цене
  • Розничная сумма – сумма товаров по розничной цене, указанной в накладной

Список платежей

В списке платежей отображаются факты оплаты по документу в пользу поставщика.

В списке отображаются столбцы:

  • Дата/время совершения операции выплаты
  • Сумма оплаты
  • Денежный счет, с которого была сделана выплата
  • Комментарий к платежу
  • Сотрудник, выполнивший платеж

Если для накладной на поступление не добавлены платежи, то она будет считаться неоплаченной. Задолженность перед поставщиком будет отображаться в журнале поступлений и отчете по взаиморасчетам в мастере отчетов.

Добавить

Кнопка "Добавить" Нажмите кнопку, чтобы добавить платеж к накладной

При нажатии на кнопку "добавить" будет открыто окно для добавления платежа к документу "накладная"

В открывшемся окне укажите сумму, выберите счет и нажмите "снять", чтобы добавить платеж по накладной.

Опция "нефискальная операция" доступна для платежей для счетов с типом "денежный ящик" и включенной в настройках опции "Разрешить нефискальные операции". В случае включения такой опции снятие наличных из денежного ящика не будет фиксироваться в ККМ.

Удалить

Кнопка "Удалить" Нажмите кнопку, чтобы удалить платеж из списка

Выберите один или несколько платежей, чтобы удалить их из списка.

Строка итогов для списка платежей

Строка итогов содержит следующую информацию:

  • Платежей – кол-во платежей в списке
  • Оплачено – общая сумма платежей в списке
  • Задолженность – разница между общей суммой товаров по накладной и внесенными платежами

Сохранить

Нажмите кнопку "Сохранить", чтобы внесенные изменения в документе сохранились в базу данных.

Отменить

Нажмите "Отмена", чтобы отменить изменения, которые были внесены в документ. Программа запросит подтверждение на отмену.

Вопросы и ответы

Есть ли автоматическая нумерация накладных?

Для того, чтобы входящей накладной (приходу) автоматически присваивался номер, не заполняйте поле Номер в карточке накладной (оставьте его пустым). После сохранения такой накладной ей будет присвоен номер автоматически.

Как установить закупочную цену на товар?

Закупочная цена на товар – это свойство документа "поступление" (накладная), а не самого товара. Т.е. для того, чтобы установить закупочную цену на какой-то товар, необходимо создать документ "Поступление".

В меню главной формы нажмите Документы – Новое поступление. В карточке накладной вы сможете добавить необходимые товары и указать для них розничные, закупочные цены и количество.

Можно ли в программе создать черновик накладной, а принять ее позже?

Да, для накладной есть опция "Еще в пути".

Если у накладной включена опция "Еще в пути", то при ее сохранении остатки для товаров не будут изменены. Т.е. это "черновик" документа. Это может быть необходимо, когда вы уже подготовили документ, но товар еще фактически не поступил в торговую точку.

Как поставить товар на приход?

Чтобы поставить товар на приход, необходимо создать накладную (приход) с товарами. Сделать это можно двумя способами:

  • вручную заполнить карточку накладной
  • загрузить накладную из документа Excel.

Как установить наценку на товар?

Наценка на товар происходит в карточке накладной в процессе ее оформления.

Чтобы поле "наценка" отображалось, включите опцию "Показывать наценку в карточке накладной" в Файл – Настройки – Внешний вид.

Кроме того, может быть использовано автоматическое ценообразование. Процесс настройки автоматического ценообразования описан в статье базы знаний.
